# ST-research
ST-research 是一个面向学习、科研和技术检索的本地 AI 搜索助手。  它结合本地大模型与 SearXNG 元搜索能力，可以自动理解用户问题、检索网页资料、过滤低质量结果，并根据来源权重和相关性生成结构化回答。相比普通搜索引擎，ST-research 更关注“答案从哪里来”；相比普通聊天机器人，它更强调可验证来源、证据强度和不确定性说明。  适合用来查论文、找技术资料、理解复杂概念、验证模型回答，以及快速整理可靠参考信息。

使用说明：
1. 如果你是第一次使用并且没有安装过任何依赖和模型，你只需要打开install_dependence.bat，然后一直输入“Y”并点击回车键就好，它会帮你安装好python环境、依赖以及默认模型（qwen3.5-2B）
2. 使用前请确保你已经下载了docker并且已经打开
3. 双击start.bat等待启动完成即可使用
4. 你可以通过修改cfg.txt里的MODEL_PATH来更换模型路径
5. website.txt 中你可以自定义信息源的权重，其中权重的值为1~5，值越大表示越重视这个信息源（例如：a.com 5 表示a网站的权重为5）

注意：
1. 使用前请确保你的设备能够运行对应的模型
2. 默认配置下需要6G及以上的磁盘空间（其中模型qwen3.5-2B空间占比最高）
3. 如果你pip使用的是中国镜像源，请注意分辨默认下载的torch是否为GPU版本
